# CustomCodeTestEnv
C# Custom Code test environment of Power Apps Custom Connector.

Power Appsのカスタムコネクタに登場したC#カスタムコードの開発・テスト環境。
.NET Core C#8.0の環境。

Script.csの内容をカスタムコードに貼り付けて保存します。
